Journal-Quality Chart Styles

Each style replicates the actual figure requirements of its target journal family.
Specs are sourced from official author guidelines. All palettes are colorblind-safe (Okabe-Ito or journal-approved).

Vacancy Profile (3 curves: Total, Ions, Recoils)

APS Physical ReviewRecommended

Font: Helvetica Neue Condensed (we use Helvetica/Arial)
Labels: 10pt axis labels, 9pt tick numbers
Axes: 0.5pt lines, ticks inward, minor ticks visible
Grid: None  |  Lines: 1.0pt data lines
Units: Parentheses — "Depth (µm)" not "[µm]"

Nature Nature / Nature Materials

Font: Helvetica or Arial, 5–7pt at print size
Axes: 0.25–1pt range, thin clean lines
Grid: None  |  Lines: 0.5–1pt
Color: Okabe-Ito palette (colorblind-safe)
Style: Open frame (L-shape, no top/right border)

AAAS Science

Font: Times New Roman / serif, 6–8pt
Axes: Thin black, ticks inward
Grid: None  |  Lines: 1pt
Color: Muted, B&W distinguishable (line styles)
Style: Box frame, compact, dense

Elsevier Acta Materialia / Scripta

Font: Arial / Times New Roman
Axes: Medium weight, ticks outward
Grid: Light gray horizontal lines
Lines: 1.5pt, colored with dash differentiation
Style: Clean with subtle grid, open frame

IEEE IEEE Transactions

Font: Times New Roman, 8pt
Axes: 0.5pt, ticks inward, minor ticks
Grid: None  |  Lines: 0.8pt
Color: Must be readable in B&W print
Style: Box frame, markers on lines, compact

Springer JMR / J. Mater. Sci.

Font: Arial or Helvetica, 8–10pt
Axes: Medium black, ticks inward
Grid: None  |  Lines: 1.5pt
Color: Colorblind-safe, distinct styles
Style: Box frame, bold axis labels, serif legend
